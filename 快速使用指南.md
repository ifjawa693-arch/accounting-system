# 📖 财务会计系统 - 快速使用指南

## 🌐 访问系统

打开浏览器访问：**http://localhost:5173/**

---

## 🎯 核心功能演示

### 1️⃣ 添加客户（推荐首先尝试）

1. 点击左侧菜单 **业务管理** → **客户管理**
2. 点击右上角 **添加客户** 按钮
3. 填写表单：
   - 客户名称：测试客户公司
   - 联系人：张经理
   - 联系电话：13800138000
   - 电子邮箱：test@example.com
   - 地址：北京市朝阳区
   - 应收账款：50000
4. 点击 **添加客户** 按钮保存
5. ✅ 新客户立即出现在列表中

**试试其他操作：**
- 点击蓝色 ✏️ 编辑按钮修改客户信息
- 点击红色 🗑️ 删除按钮删除客户
- 在搜索框输入关键词搜索客户

---

### 2️⃣ 创建会计分录

1. 点击左侧菜单 **总账会计** → **做分录**
2. 填写凭证信息：
   - 凭证日期：选择今天
   - 凭证号：系统自动生成
3. 填写第一行（借方）：
   - 会计科目：1002-银行存款
   - 借方金额：10000
   - 摘要：销售收入
4. 填写第二行（贷方）：
   - 会计科目：6001-主营业务收入
   - 贷方金额：10000
   - 摘要：销售收入
5. 查看底部合计：
   - ✅ 如果显示"借贷平衡，可以保存"则可以提交
   - ❌ 如果显示差额，需要调整金额
6. 点击 **保存凭证** 按钮

**高级功能：**
- 点击 **添加分录行** 添加更多行
- 点击 🗑️ 删除不需要的分录行（至少保留2行）

---

### 3️⃣ 管理员工信息

1. 点击 **业务管理** → **员工管理**
2. 点击 **添加员工** 按钮
3. 填写员工信息：
   - 员工姓名：李明
   - 职位：会计
   - 部门：财务部（下拉选择）
   - 联系电话：13900139000
   - 电子邮箱：liming@company.com
   - 薪资：10000
   - 入职日期：选择日期
4. 点击 **添加员工** 保存

---

### 4️⃣ 费用报销审批

1. 点击 **业务管理** → **费用管理**
2. 点击 **提交费用** 按钮
3. 填写费用信息：
   - 费用日期：选择日期
   - 报销人：选择员工
   - 费用类型：差旅费
   - 费用金额：1000
   - 费用说明：出差上海
4. 点击 **提交费用**

**审批操作：**
- 在列表中找到"待审核"状态的费用
- 点击 **批准** 同意报销
- 或点击 **拒绝** 驳回申请
- 状态立即更新

---

### 5️⃣ 查看财务报表

1. 点击 **总账会计** → **会计报表**
2. 选择报表类型：
   - 资产负债表
   - 利润表
   - 现金流量表
3. 选择日期范围
4. 点击 **生成报表**
5. 查看详细的财务数据

## 💡 使用技巧

### 🔍 搜索功能
大多数列表页面都有搜索框：
- 客户管理：搜索客户名称或联系人
- 供应商管理：搜索供应商名称或联系人
- 员工管理：搜索姓名、职位或部门
- 会计科目：搜索科目名称或编码

### 📊 筛选功能
部分页面支持按状态筛选：
- 采购订单：全部状态/待审核/已审核/已完成
- 销售单：全部状态/草稿/已发送/已收款
- 费用管理：全部状态/待审核/已批准/已拒绝

### 🎯 表单验证
- 标记 * 的字段为必填项
- 提交前系统会自动验证
- 确保所有必填项都填写完整

### 🔔 操作反馈
- 删除操作会弹出确认对话框
- 成功保存后弹窗会自动关闭
- 列表数据立即更新显示

## 🆘 常见问题

**Q: 点击按钮没有反应？**
A: 请确保开发服务器正在运行，检查浏览器控制台是否有错误。

**Q: 表单无法提交？**
A: 检查是否填写了所有标记 * 的必填项。

**Q: 刷新后数据丢失？**
A: 这是正常的，当前版本数据保存在内存中，刷新会重置。

**Q: 如何保存数据？**
A: 当前版本暂不支持持久化，后续版本会添加。

**Q: 可以同时打开多个弹窗吗？**
A: 不可以，每次只能打开一个弹窗。关闭当前弹窗后才能打开新的。
